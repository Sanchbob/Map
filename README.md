# map

1. Мета проекту:
Створити інтерактивну web мапу КПІ, що вміє масштабувати свій інтерфес на різну роздільну здатність екранів (ПК, планшети, смартфони, ін.).


2. Використовуванні технології:

Program language:
JavaScript

Database:
NoSQL

Frameworks:
AngularJS
Foundation

Platform:
NodeJS

Server:
Sails
